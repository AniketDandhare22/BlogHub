## ✨ Key Features

- **🎨 AI Image Generator:** Integrated **Hugging Face (Stable Diffusion)** and **DALL·E** to let users generate custom cover art for their blogs instantly.
- **🤖 AI Content Assistant:** Uses Google Gemini API to auto-generate blog summaries, outlines, and content ideas.
- **🔐 Secure Authentication:** Robust JWT-based auth with HTTP-Only cookies and Google OAuth 2.0 integration.
- **💳 Monetization:** Razorpay payment gateway integration for "Pro" membership subscriptions and credit management.
- **🌗 Dark/Light Mode:** Fully responsive UI with persistent theme switching.
- **⚡ Performance:** Implemented Skeleton Loaders and Optimistic UI for a seamless experience.
- **📊 User Dashboard:** Analytics for post views, "Recent Reads" history, and profile management.
- **📝 Rich Media:** Support for Markdown-based articles and video streaming.

## 🚀 Getting Started Locally

Follow these steps to run the project on your local machine.

### 1. Clone the Repository
```bash
git clone https://github.com/AniketDandhare22/BlogHub.git
cd BlogHub
cd "blog (frontend)"
npm run dev
cd ..
cd backend
npm start
```
